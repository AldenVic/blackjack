== DESCRIPTION:

A text-based game of blackjack made using Ruby

== FEATURES/PROBLEMS:

* Only primitive version so far. No splitting, no insurance.

== REQUIREMENTS:

* rubygems

== INSTALL/RUN:

* Make sure ruby is installed
* Go to the app's bin direcory (~/blackjack/bin)
* run :
	$ ./play_blackjack
* Note: In case the file fails to run, convert it to an executable using
	$ sudo chmod +x play_blackjack
  and then try running it again
